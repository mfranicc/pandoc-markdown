# Markdown
Markdown je lagani jezik za oblikovanje teksta koji omogućuje pisanje čitljivih dokumenata, README datoteka, dokumentacija i wiki stranica
<br>
## Osnovna sintaksa
Koristimo jednostavne znakove poput `#`, `*`, `-` i `` ` `` za strukturiranje i formatiranje teksta.

### Naslovne razine
Ako želim koristiti naslovne razine, potrebno je unijeti sljedeću sintaksu
```bash
# Naslov 1
## Naslov 2
### Naslov 3
#### Naslov 4
```

### Podebljavanje i kurziv
*kurziv* 
**podebljano**  
***podebljano i kurziv***

### Liste
Nenumerirana lista:
- prvo
- drugo
- treće

Numerirana lista:
1. prva
2. druga
3. treća
